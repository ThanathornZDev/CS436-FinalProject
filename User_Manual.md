# คู่มือการใช้งานระบบ BMI Tracker

## 1. การเข้าใช้งานระบบ
1. เปิด Web Browser (แนะนำ Google Chrome)
2. เข้าไปที่ลิงก์ระบบ: `https://<your-username>.github.io/bmi-tracker/` (ตัวอย่าง)

## 2. การคำนวณ BMI
1. กรอกส่วนสูงในช่อง "ส่วนสูง (cm)" (เช่น 170)
2. กรอกน้ำหนักในช่อง "น้ำหนัก (kg)" (เช่น 65)
3. กดปุ่ม "คำนวณ BMI"
4. ดูผลลัพธ์ในกล่องผลลัพธ์:
   - ค่า BMI
   - สถานะ (น้ำหนักน้อย / ปกติ / น้ำหนักเกิน / อ้วน)

## 3. การดูประวัติการวัด
- เลื่อนลงมาที่ส่วน "ประวัติการวัด"
- ระบบจะแสดงตารางรายการวันที่วัด น้ำหนัก ส่วนสูง ค่า BMI และสถานะ

### การลบประวัติเป็นรายการ
1. เลือกแถวที่ต้องการลบ
2. กดปุ่ม "ลบ" ในคอลัมน์ "จัดการ"
3. ยืนยันการลบ (หากมี popup)

### การล้างประวัติทั้งหมด
1. กดปุ่ม "ล้างประวัติทั้งหมด"
2. กดยืนยันเมื่อมีข้อความถามยืนยัน
3. ตารางประวัติจะว่างเปล่า

## 4. การดูกราฟ BMI
- เลื่อนลงมาที่ส่วน "กราฟแนวโน้ม BMI"
- กราฟเส้นจะแสดงค่า BMI ตามวันที่วัด ช่วยให้เห็นแนวโน้มเพิ่ม/ลด

## 5. ข้อควรระวัง
- ข้อมูลถูกเก็บไว้ใน Browser ปัจจุบัน ถ้าเปลี่ยนเครื่องหรือเปลี่ยน Browser ข้อมูลจะไม่ตามไป
- ถ้ากดล้างประวัติทั้งหมด ข้อมูลเดิมจะหายและไม่สามารถกู้คืนได้
