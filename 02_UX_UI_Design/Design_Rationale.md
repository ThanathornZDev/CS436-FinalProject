# BMI Tracker – Design Rationale

## 1. UX Concept
ระบบออกแบบแบบ Single Page เพื่อความง่ายในการใช้งาน ผู้ใช้สามารถคำนวณ ดูผล คำแนะนำ ประวัติ และกราฟ ได้ในหน้าเดียว

## 2. Layout
1. ส่วนหัว: ชื่อระบบ  
2. ฟอร์มกรอกข้อมูล + ปุ่มคำนวณ  
3. แสดงผลลัพธ์ BMI  
4. ส่วนคำแนะนำด้านการออกกำลังกายและโภชนาการ  
5. ตารางประวัติ  
6. กราฟแนวโน้ม BMI

## 3. UI สีและธีม
- พื้นหลังโทนอ่อนเพื่อความอ่านง่าย  
- สีฟ้า/เขียวเพื่อสื่อถึงสุขภาพ  
- สีสถานะแบบเข้าใจง่าย  
  - ฟ้า = Underweight  
  - เขียว = Normal  
  - ส้ม = Overweight  
  - แดง = Obese  

## 4. Forms
- แบบฟอร์มเน้นความง่าย  
- Placeholder เพื่อช่วยผู้ใช้  
- แจ้ง error ชัดเจนในกรณีกรอกผิด

## 5. จะใช้ Chart.js เพื่อวาดกราฟ
เพราะใช้งานง่าย โหลดผ่าน CDN ได้
