# User Manual – BMI Tracker

## 1. เปิดใช้งาน 
เปิดเว็บ : https://bmi-tracker-seven.vercel.app/

## 2. คำนวณ BMI
1. กรอกส่วนสูง (cm)  
2. กรอกน้ำหนัก (kg)  
3. กด “คำนวณ BMI”  
4. ดูผลลัพธ์และคำแนะนำสุขภาพ

## 3. จัดการประวัติ
- เลื่อนลงไปที่ส่วนประวัติ  
- ปุ่มลบ = ลบรายรายการ  
- ปุ่มล้างประวัติทั้งหมด = ลบทั้งหมด

## 4. ดูกราฟ BMI
- เลื่อนลงไปด้านล่างสุด  
- กราฟจะแสดงแนวโน้มตามวัน

## 5. หมายเหตุ
- ข้อมูลผูกกับ Browser เครื่องเดียว (LocalStorageKey)
- หากล้างประวัติ ข้อมูลจะหายถาวร
