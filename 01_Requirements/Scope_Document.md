# BMI Tracker – Scope Document  
*Final Project – CS436 Software Engineering*

## 1. Objective (วัตถุประสงค์)
ระบบ BMI Tracker ถูกพัฒนาขึ้นเพื่อช่วยให้ผู้ใช้สามารถ:
- คำนวณค่า BMI จากน้ำหนักและส่วนสูง
- ติดตามประวัติ BMI ของตนเองในแต่ละครั้ง
- ดูแนวโน้มการเปลี่ยนแปลงของ BMI ผ่านกราฟ
- ได้รับคำแนะนำด้านการออกกำลังกายและโภชนาการที่เหมาะสมตามค่า BMI

ระบบนี้เป็น Web Application แบบ Static ใช้ HTML/CSS/JavaScript และเก็บข้อมูลด้วย LocalStorage

---

## 2. In-Scope (อยู่ในขอบเขต)
1. กรอกข้อมูลส่วนสูง (cm) และน้ำหนัก (kg)
2. ตรวจสอบความถูกต้องของ input
3. คำนวณ BMI และแสดงสถานะ เช่น
   - Underweight  
   - Normal  
   - Overweight  
   - Obese
4. แสดงคำแนะนำด้านการออกกำลังกายและการกิน ตามช่วง BMI
5. เก็บบันทึกประวัติ BMI ลง LocalStorage
6. แสดงประวัติในรูปแบบตาราง
7. ลบประวัติรายรายการ
8. ลบประวัติทั้งหมด (Clear All)
9. แสดงกราฟแนวโน้ม BMI (Chart.js)
10. โหลดข้อมูลประวัติอัตโนมัติเมื่อเปิดเว็บ

---

## 3. Out-of-Scope (นอกขอบเขต)
- ระบบ Login / การรองรับหลายผู้ใช้
- Database จริง / Cloud Storage
- แจ้งเตือนผ่านมือถือหรือ LINE
- การวิเคราะห์สุขภาพเชิงลึกที่ซับซ้อน
- Export PDF/CSV
- ระบบหลายภาษา

---

## 4. Constraints (ข้อจำกัด)
- ใช้เฉพาะ Client-side (ไม่มี Backend)
- ข้อมูลผูกกับ Browser/เครื่องเดียว
- ปริมาณข้อมูลประวัติขึ้นอยู่กับ LocalStorage

---

## 5. Target Users (ผู้ใช้เป้าหมาย)
- นักศึกษา
- ผู้ที่ต้องการควบคุมน้ำหนักและติดตาม BMI
- ผู้ที่ต้องการคำแนะนำสุขภาพแบบพื้นฐาน

---

## 6. Definitions (คำจำกัดความ)
- **BMI (Body Mass Index)**: ดัชนีมวลกาย  
- **LocalStorage**: พื้นที่เก็บข้อมูลใน Browser  
- **Record**: รายการบันทึกหนึ่งครั้ง (วันที่/น้ำหนัก/ส่วนสูง/BMI/สถานะ)
